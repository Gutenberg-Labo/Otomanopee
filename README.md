## Otomanopee について

オープンソースフォント「GL-Otomanopee」の Google Fonts 提供バージョンです。  
漫画の描き文字をイメージした和文仮名文字フォントです。  
本家配布版との違いは小学1年生および小学2年生で習う漢字240文字を収録しています。
